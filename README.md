# isopaint

##isometric doodlin'.

leftclick/rightclick: cycle through shades

click&drag: drag the color around

a: shrink triangles

z: grow triangles

r: color (r)ed

g: color (g)reen

b: color (b)lue

w: black and (w)hite

s: save as png

?: hide/display this help

resize the window at will.
